DOM stands Document Object Model - it is a node-tree representation of all the elements within our HTML page.
It allows us to identify and access these elements and then use JS functions to apply behavior to them (that's the manipulation part).


There are various ways to get an element:

  • id - (using getElementById function)
  • name - (using getElementByName function)
  • class - (using getElementByClass function)
  • There are others too...
