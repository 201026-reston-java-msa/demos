Intro to CSS

We have a pretty good understanding of HTML, but all of our pages look kinda plain. The browser applies default styling such as strong, which makes text bold. But how do we make it look nicer ?

CSS stands for Cascading Style Sheets. There are 3 ways to include CSS in an html page:

  1. Inline (with the style attribute)
  2. Using internal stylesheets (using the style element in the head of the doc)
  3. Using external stylesheets (we link to an external .css document)
Css rules are key-value pairs, separated by a colon ( : ). There are many rules that we can apply. One example is color

We are able to sleect elements by more than just their tag. We can give particular elements special ID's or a particular class. Classes are selected by a . and id's are selected by a #.
